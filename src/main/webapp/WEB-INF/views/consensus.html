Consensus 변동(2021/12(E) 기준)

각 증권사의 애널리스트들은 리포트를 통해서 연간, 분기별 컨센서스를 예측합니다.

이 지표들은 실적 발표를 하기 전까지 변동이 가능한 지표로, 실적이 상승하거나 하향될 것으로 예측될 때 이 방향성을 체크할 수 있다면 적절한 투자 타이밍을 잡을 수 있을 것입니다.

또한 신규로 컨센서스가 나오는 경우, 즉 증권사에서 신규로 커버리지하는 경우 해당 종목은 향후 좋은 주가흐름을 나타낼 확률이 높습니다.

각 투자자들은 컨센서스의 변화가 나오는 주식들을 잘 모니터링하여 수익률을 극대화하기 바랍니다. 해당 데이터는 매월 초 컨센서스 대비 변경된 내역을 매일 집계하여 보여 줍니다.


Analysts of each securities firm use reports to forecast annual and quarterly consensus.

These indicators are subject to change until the performance is announced, and if you can check this direction when performance is expected to rise or fall, you will be able to set the appropriate investment timing.

In addition, if a new consensus comes out, that is, if a securities company newly covers it, the stock is likely to show good share price flow in the near future.

Each investor is encouraged to maximize returns by monitoring stocks where the consensus changes. This data shows daily aggregated changes against the consensus at the beginning of each month.
